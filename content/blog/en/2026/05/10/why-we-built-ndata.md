# Why We Built N(DATA)

Finding clean technical specs for a device should be easy. It is not.

Most spec sites are full of ads, pop-ups, and slow load times. The data is often incomplete, inconsistently formatted, or locked behind a login. Comparing two devices means opening five tabs and switching between them manually.

We wanted something different: a fast, clean, open archive that puts the data first and everything else out of the way.

## The Design Philosophy

N(DATA) takes design cues from Nothing Technology, a company that strips away what is not needed and keeps only what matters. That same idea drives this project. No accounts. No ads. No paywalls. Just a JSON file and an interface that gets out of your way.

## Open by Default

Everything in N(DATA) is open. The data is a plain JSON file you can fetch directly. The source code is on GitHub under the MIT license. If you want to build something on top of it, you can.

## What Comes Next

We are working on growing the archive, improving the data quality, and adding tools for contributors. If you want to help, the report page is the easiest starting point.

## v1.11.0 — 2026-05-10

### New Features

- **Shared Layout** — Navbar and footer now injected from a single `layout.js` file, consistent across every page
- **Content Repo** — Blog, news, and updates load live from public NData-Content GitHub repo as Markdown
- **Bookmarks** — Save items locally, persists across sessions
- **Compare** — Side-by-side spec comparison for up to 4 items
- **Sort** by price (asc/desc) and date added (new/old)
- **Card image previews** with halftone shader effect and auto-scroll on hover
- **Copy Specs** button on each card
- **Keyboard shortcut** — press `/` to focus search bar
- **Multi-language support** — English and Arabic with full RTL layout
- **EUR and GBP currencies** added alongside USD, SAR, JPY
- **New categories** — Gaming, Monitors, Networking
- **Auto ID assignment** (prefix-NNNNNN format)
- **Full API docs** page at `/api/docs`
- **Live API explorer** at `/api/api` with filter, sort, download
- **Bookmark button** in product modal header, synced with card state

### Fixes

- Theme toggle icon no longer disappears after re-render
- Burger menu icon stays consistent across state changes
- All icon elements use `data-theme-icon` attribute for reliable updates
- Currency display now consistent across all pages via `layout.js`
- Language labels and dropdowns consistent across all pages

### Content

- Blog, news, and updates now fetched live from NData-Content repo
- Arabic translations completed for all UI strings
- Footer description and links fully translated

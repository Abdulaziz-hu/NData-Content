## v1.11.0 — 2026-05-10

### ميزات جديدة

- **تخطيط مشترك** — يتم الآن حقن شريط التنقل والتذييل من ملف `layout.js` واحد، متسق عبر كل صفحة
- **مستودع المحتوى** — تُحمَّل المدونة والأخبار والتحديثات مباشرة من مستودع NData-Content العام على GitHub
- **المحفوظات** — حفظ العناصر محلياً مع الاستمرارية عبر الجلسات
- **المقارنة** — مقارنة المواصفات جنباً إلى جنب لما يصل إلى 4 عناصر
- **الفرز** حسب السعر (تصاعدي/تنازلي) وتاريخ الإضافة
- **معاينات صور البطاقة** مع تأثير النقاط المنقوطة وتمرير تلقائي عند التحويم
- **زر نسخ المواصفات** على كل بطاقة
- **اختصار لوحة المفاتيح** — اضغط `/` للتركيز على شريط البحث
- **دعم متعدد اللغات** — العربية والإنجليزية مع تخطيط RTL الكامل
- **عملات EUR وGBP** مضافة إلى جانب USD وSAR وJPY
- **فئات جديدة** — الألعاب، الشاشات، الشبكات
- **تعيين معرف تلقائي** (بتنسيق prefix-NNNNNN)
- **صفحة توثيق API كاملة** على `/api/docs`
- **مستكشف API مباشر** على `/api/api`

### إصلاحات

- أيقونة تبديل السمة لم تعد تختفي بعد إعادة العرض
- أيقونة القائمة المتنقلة تبقى متسقة عبر تغييرات الحالة
- العملة والترجمة متسقة عبر جميع الصفحات عبر `layout.js`
- تمت ترجمة جميع نصوص واجهة المستخدم للعربية
